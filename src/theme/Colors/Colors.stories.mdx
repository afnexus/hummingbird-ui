# README :

This story book consists of two color themes, Light Theme and Dark Theme. Specific Colors can be used anytime with the right name.  
In order to switch between themes, user can use useState hooks and remember to wrap full code in ThemeProvider and CssBaseline.  
E.g.<br /><br />
import React, {useState} from 'react';  
import { ThemeProvider } from '@mui/material';  
import { lightTheme,darkTheme } from '../theme';  
import { CssBaseline } from '@mui/material';  <br /><br />
export const LightDark = () => {**
    const [darkMode, setDarkMode] = useState(false);**   
  return (
    <ThemeProvider theme={darkMode ? darkTheme : lightTheme}>
        <CssBaseline />
        <ButtonWithMUI color='primary' text='Color'/>
        <button onClick = {() => setDarkMode (!darkMode)}>switch</button>
    </ThemeProvider>
  )
}
